# 📌 Componente re direct landing eventos

## Descripción

Mediante este componente se podrá redirigir de una URL a otra URL. Por ej: redirigir de la home a una landing (de manera automática).

### Pasos para la coinfiguración

1.  Ingresar a la home o sábana que se quiera configurar. Para este ejemplo usaremos **/newin.**<br>

    <figure><img src="../../.gitbook/assets/2 (6).png" alt=""><figcaption></figcaption></figure>
2.  Abrir el bloque **Layout flexible para resultados de búsquedas personalizadas**,  abrir el bloque **Resultado de la búsqueda** (desktop o mobile) e ingresar a **Componente Redirect Page**. <br>

    <figure><img src="../../.gitbook/assets/3 (3).png" alt=""><figcaption></figcaption></figure>
3.  Al ingresar, tendremos la opción de **encender o apagar el componente** y el campo llamado **Link a redireccionar** donde debemos completar la URL que queremos configurar para la redirección. En este caso se coloca solo “/” para indicar que tiene que redirigir a la Home.<br>

    <figure><img src="../../.gitbook/assets/image (81).png" alt=""><figcaption></figcaption></figure>
4. Al terminar de configurar, hacemos click en **Guardar** para que apliquen los cambios.

{% hint style="info" %}
En el caso de la home, el bloque Redirect page se encuentra fuera del layout de desktop y mobile por lo que sólo hay que configurarlo una vez.
{% endhint %}
