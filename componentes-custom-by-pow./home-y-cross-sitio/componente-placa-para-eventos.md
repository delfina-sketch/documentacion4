# 📌 Componente Placa para eventos

### Descripción

Mediante este componente se podrá colocar una placa la cual se mantendrá visible en todo el sitio web, sin permitir que los usuarios naveguen en el.

## Configuración

Para configurar la Placa, se debe encender en el site editor, tanto en la Home como en sábana.

### Paso a paso para agregar la tabla de talles

1.  Acceder al Site editor -> Componente Menu - Bloques Header -> lo desplegamos y entramos en "Close Page"

    <figure><img src="../../.gitbook/assets/Captura de pantalla 2024-10-25 a la(s) 9.44.00 a. m..png" alt=""><figcaption></figcaption></figure>
2.  En el componente lo único que se deberá tocar es "Encender/apagar componente" y selecionar Guardar.

    <figure><img src="../../.gitbook/assets/Captura de pantalla 2024-10-25 a la(s) 9.45.52 a. m..png" alt=""><figcaption></figcaption></figure>
3.  De la misma forma, tenemos que activar la placa en las sábanas. Se deberá encender en una sola. Para esto vamos a una sábana en el site editor y repetimos los pasos. Vamos a "Componente Menú - Bloques Header" -> "Close Page":

    <figure><img src="../../.gitbook/assets/Captura de pantalla 2024-10-25 a la(s) 9.50.04 a. m..png" alt=""><figcaption></figcaption></figure>
4.  Y adentro del componente lo encendemos y guardamos:

    <figure><img src="../../.gitbook/assets/Captura de pantalla 2024-10-25 a la(s) 9.51.46 a. m..png" alt=""><figcaption></figcaption></figure>
5. En el componente van a ver que figura un parámetro: ?view-production -> el mismo se utiliza para poder ingresar a la web mientras se encuentra cerrada. El parámetro lo van a colocar al final de la URL [https://www.47street.com.ar/?view-production](https://www.47street.com.ar/?view-production)
