# 💻 Tamaño de imágenes

{% hint style="warning" %}
Las medidas están expresadas al doble para que en pantallas de retina no se vea pixelado
{% endhint %}

{% hint style="danger" %}
Recordar que los pesos sugeridos por VTEX para la carga de imágenes (de producto, banners, etc), videos y GIF es de 300 KB. \
**Superar este límite implica una reducción en la correcta performance del sitio**.&#x20;
{% endhint %}
